# 🏙️ BrickView Pro: Real Estate Analytics

An interactive dashboard built with **Python, SQL, and Streamlit** to explore market trends, agent performance, and buyer behavior.

## 🚀 Features
- **Interactive Map:** Geospatial distribution of properties with hover insights.
- **Advanced Filtering:** Multi-city and price-range search engine.
- **SQL FAQ:** 30+ business queries categorized by domain.
- **CRUD Operations:** Full database management capabilities.

## 🛠️ Tech Stack
- **Frontend:** Streamlit, Plotly, Folium
- **Backend:** Python (Pandas), SQLite3
- **Data Source:** JSON/CSV (162+ Credits coursework data)

## 🏃 How to Run
1. Clone the repo: 'https://github.com/aryaman2213/BrickView-Real-Estate-Analytics-Platform'
2. Install dependencies: `pip install -r requirements.txt`
3. Run migration: Execute cells in `BrickView_Analysis.ipynb`
4. Launch app: `streamlit run app.py`
